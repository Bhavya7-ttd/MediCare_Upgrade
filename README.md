# 💊 MediCare – AI-Powered Family Medicine Management System

## Overview

MediCare is a healthcare management platform that helps patients, elderly individuals, and caregivers manage medicines, appointments, prescriptions, doctors, and health records.

## Features

* 💊 Medicine Reminder System
* 🎤 Voice-Based Medicine Entry
* 🤖 AI Health Assistant
* 👨‍👩‍👧 Family Member Management
* 📅 Appointment Management
* 🏥 Doctor Management
* 📄 Prescription Records
* 👨‍⚕ Caregiver Dashboard
* 🚨 Emergency SOS
* 📊 Analytics Dashboard
* 💧 Water Reminder
* ⚖ BMI Calculator
* 🔔 Notification Center

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Firebase Authentication
* Cloud Firestore
* Firebase Hosting
* Gemini AI
* Web Speech API
